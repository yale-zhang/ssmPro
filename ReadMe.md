#### SSM project 
   SpringMVC+Spring+Mybatis
    - Spring (IOC DI AOP 申明式事务处理)   
        * 采用aop 的方式处理参数问题
    - SpringMVC（支持Restful风格）     
    - Hibernate Validate（参数校验）  
        * 自定义validator标签(和hibernate validator组合使用)   
    - Redis 缓存
        * Spring-data redis
        * 采用Jedis Cluster 
    - Quartz时间调度
